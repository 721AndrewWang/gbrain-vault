# 常用命令

MOYAG项目开发常用命令速查。

## Hermes Agent

```bash
# 查看配置
hermes config | head -20
hermes doctor

# 技能管理
hermes skills list
hermes skills search <keyword>

# Cron管理
hermes cron list
hermes cron run <job-id>

# 会话管理
hermes sessions list
hermes sessions browse

# MCP管理
hermes mcp list
hermes mcp test gbrain

# 模型切换
hermes model
```

## Claude Code

```bash
# 直接exe路径 (bash wrapper有bug)
CLAUDE="$HOME/AppData/Roaming/npm/node_modules/@anthropic-ai/claude-code/bin/claude.exe"

# MCP管理
"$CLAUDE" mcp list
"$CLAUDE" mcp add -s user <name> -- <command>

# 查看配置
cat ~/.claude/settings.json
cat ~/.claude.json

# Hook脚本
ls ~/.claude/hooks/
```

## GBrain

```bash
# 索引全部笔记
cd ~/Documents/GBrain\ Vault
python gbrain.py index

# 搜索
python gbrain.py search "关键词"

# 监听模式 (自动索引)
python gbrain.py watch

# 统计
python -c "from gbrain import GBrain; gb=GBrain('.'); print(gb.stats())"
```

## Git

```bash
# 知识库备份
cd ~/Documents/GBrain\ Vault
git add *.md */*.md
git commit -m "backup: $(date -I)"
git push
```

## 维护

```bash
# 磁盘检查
python ~/AppData/Local/hermes/scripts/disk-watchdog.py

# 清理旧会话 (手动)
hermes sessions prune --older-than 30

hermes cron list --all
```
